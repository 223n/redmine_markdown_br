# Redmine Markdown BR Plugin

[![GitHub](https://img.shields.io/badge/license-Apache%20Version%202.0-blue.svg)](LICENSE)

## 使い方

説明やWikiなどの改行したい場所で ```{{br}}``` と入力してください。

そこで改行されます。
